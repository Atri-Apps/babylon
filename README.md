# Stream video and review logs 

BabylonAI makes it easy to log raw input and prediction data

## How to use this app?

### Step 1: Setup

```shell
git clone https://github.com/Atri-Apps/babylon.git
```

```shell
cd babylon
```

```shell
pipenv install
```

### Step 2: Start the editor

```shell
pipenv shell
```

```shell
atri start
```
